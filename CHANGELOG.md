# Changelog

## 0.1.0

- Initial release
- Multi-timeframe candle display as a Lightweight Charts series primitive
- Configurable timeframe columns (any timeframe string)
- Built-in OHLC aggregation: 4H, 8H, 12H, 1D, 1W, 1M, 3M, 6M, 1Y
- Per-column color customization (up/down body and wick)
- Separator lines and labels between columns
- Background color support
- Adjustable candle width, spacing, and column gaps
